# Task 05: Deceased Agent Runtime

## Goal
単なる会話ではなく、クエスト進行と合意形成を導くAI Agentを実装する。

## Tasks
- [x] Agent設計
  - Persona（口調/価値観/目的）
  - Safetyルール（違法/危険提案の拒否）
- [x] Planner実装
  - 目標をサブタスク化し、次アクションを提示
  - 進捗に応じて条件不足を説明
- [x] Memory実装
  - セッション記憶（プレイヤー発言、提出履歴）
  - 長期記憶（故人の意図、分配哲学）
- [x] 合意形成メッセージ
  - 4人協力が必要な理由を明示
  - 参加未完了プレイヤーへの再招集文面
- [x] Evaluation
  - 会話品質テスト（目的整合、説明一貫性、暴走抑制）

## Acceptance Criteria
- Agentが常に「次に何をすべきか」を具体提示できる
- 条件未達時に不足情報を特定して説明できる

## 人間がやること
- [ ] Agentの口調・禁止事項の最終レビュー（法務/広報観点）を行う
- [ ] 危険プロンプト一覧を作り、拒否挙動の受け入れテストを実施する
- [ ] 長期記憶の文面（故人の意図・分配哲学）を関係者合意のうえ確定する
- [ ] 本番運用での会話ログ保持期間と閲覧権限を定義する
