# Compliance Checklist (Pre-Production)

## Jurisdiction Review
- [ ] Funds transfer rules reviewed for launch countries.
- [ ] KYC/AML applicability reviewed with legal counsel.
- [ ] Game/gambling regulation applicability reviewed.

## Data & Privacy
- [ ] Proof metadata minimization policy documented.
- [ ] Retention period and deletion rules documented.
- [ ] Access logs and operator accountability defined.

## Security Controls
- [ ] Oracle production key stored in KMS/HSM.
- [ ] Emergency pause procedure tested.
- [ ] Incident response drill completed.

## Audit & Evidence
- [ ] Hash-chained audit log enabled.
- [ ] Alert feed monitored by operator.
- [ ] Weekly immutable backup process active.
